# Laravel User Management System

## Project Overview
This is a complete User Management System developed with **Laravel 11**. It features a full Admin Dashboard for web users and a comprehensive RESTful API for external integration. The system is designed with a focus on code quality, security, and testability.

## 1. Project Setup Instructions

Follow these steps to get the project running on your local machine.

### Prerequisites
- PHP 8.2 or higher
- Composer
- Node.js & NPM
- MySQL

### Installation Steps

**1. Clone and Install Dependencies**
git clone https://github.com/IvanKangJinLiang/bestweb-technologies.git
cd bestweb-technologies
composer install
npm install && npm run build
